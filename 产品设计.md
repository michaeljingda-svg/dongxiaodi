# 留学评校平台 — 产品设计文档

> 状态：内容定义阶段 | 最后更新：2026-06-24

**站名：懂校帝**

---

## 一、产品定位

**是什么：** 面向中国留学生（申请阶段 + 在读阶段）的高校真实评价社区

**不是什么：** 不是官方排名，不是中介广告，不是 USNews 的复制

**核心价值：** 学生说给学生听——真实的食堂、宿舍、教授、环境评价

**参考产品：** 虎扑（社区氛围）+ 懂球帝（分类统计）+ RateMyProfessors（评分机制）

---

---

### Tab 1：动态

类微博形式，按时间倒序

```
[头像]  用户昵称  🎓已验证在校生                    3小时前
"食堂今天的 dim sum 不错，推荐 Lobdell！"
❤️ 24   💬 5条评论   [展开]
```

规则：
- 未登录可查看
- 登录后可发布 / 点赞 / 评论
- 评论最多 2 层（评论的回复不再嵌套）
- 验证在校生（.edu 邮箱）显示 🎓 标识

---

### Tab 2：食堂

```
Lobdell Dining Hall        ⭐ 4.1  (52条)   [查看 →]
Next House Dining          ⭐ 3.7  (31条)   [查看 →]
[+ 我知道一个没列出的食堂]
```

单个食堂页：
- 食堂名 + 位置描述
- 综合评分（所有打分平均值，实时计算）
- 评价列表（时间倒序）：每条 = 1-5星 + 文字 + 点赞 + 回复
- [给这个食堂打分] → 登录后弹出评分框，提交后**不可修改**

---

### Tab 3：宿舍

结构同食堂，替换为宿舍楼：
```
Baker House      ⭐ 4.3  (41条)   [查看 →]
McCormick Hall   ⭐ 3.9  (27条)
```

---

### Tab 4：教授

```
[搜索教授姓名 ________________]

Hari Balakrishnan（计算机）    ⭐ 4.7  (33条)   [查看 →]

### Tab 5：学术 / 环境

  学术氛围          ⭐⭐⭐⭐⭐  4.6

  安全感            ⭐⭐⭐⭐   4.0

[最新评价列表]
[给学校整体环境打分]
```

## 六、用户系统

| 用户状态 | 可以做什么 |
|---------|-----------|
| 未登录 | 查看全部内容 |
| 已注册（邮箱） | 发动态、打分、写评价、点赞、回复 |
| 已验证在校生（.edu 邮箱） | 同上 + 显示 🎓 标识 |

**评分规则：提交后永久锁定，不可修改、不可删除**

## 七、已确认决策

| 决策项 | 结论 |
|--------|------|
| 目标用户 | 申请阶段 + 在读阶段，均覆盖 |
| 网站语言 | 中文优先 |
| 身份验证 | 分层：普通注册 vs 🎓 验证在校生 |
| 主页排序逻辑 | 总榜（USNews） + 分榜（地区 × 评分维度叠加） |
| 动态形式 | 类微博：短内容 + 点赞 + 评论（2层） |
| 评分是否可改 | 不可修改，永久锁定 |
| 技术路线 | 先定产品原型，技术方案后定 |
